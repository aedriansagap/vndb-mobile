# VNDB Mobile App

A beautiful, modern mobile application built with React Native (Expo) to interface with the Visual Novel Database (VNDB). This project aims to provide an improved, native mobile experience over the default website.

## Features

- **Browse Visual Novels**: Discover trending, popular, and newly released visual novels.
- **Search**: Robust search functionality filtering by tags, platforms, and length.
- **Details**: Comprehensive information for each visual novel, including cover art, description, screenshots, and character details.
- **User Integration**: Log in to view your personal lists and scores.

## Tech Stack

- Framework: React Native (Expo)
- Language: TypeScript
- API: VNDB Kana API (v2)
- Styling: React Native StyleSheet (Deep Dark Mode theme)

## Development

To run this project locally:

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npx expo start
   ```
